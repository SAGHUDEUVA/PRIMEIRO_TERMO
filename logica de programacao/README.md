# 🐍 Lógica de Programação com Python

## 📅 Conteúdo das Aulas

### 🔹 Módulo 1: Fundamentos de Python e Lógica
*   **Variáveis e Tipos de Dados:** string, int, float, boolean.
*   **Operadores:** Aritméticos, relacionais e lógicos (and, or, not).
*   **Estruturas Condicionais e de Repetição:** if, elif, else, while e for.
*   **Coleções e Funções:** Listas, dicionários e modularização de código.

### 🔹 Módulo 2: Controle de Versão com Git e GitHub
*   **Comandos Essenciais:** git init, git add, git commit.
*   **Sincronização Remota:** Criar portfólio no GitHub, usar git push e git pull.

### 🔹 Módulo 3: Boas Práticas com Clean Code
*   **Nomes Significativos:** Variáveis e funções com nomes explicativos.
*   **Regras PEP 8:** Padrões oficiais de estilo de código para Python.

## 🚀 Projetos Integradores
1.  **Simulador de Caixa Eletrônico:** Uso de condicionais, repetições e funções no console.
2.  **Gerenciador de Tarefas (To-Do List):** Uso de listas e dicionários aplicando Clean Code.
