# 🖥️ Sistemas Operacionais e Fundamentos de TI

## 📅 Conteúdo das Aulas

### 🔹 Aula 01: Introdução e Distribuições Linux
*   **Conceito Base:** O papel do S.O. como gestor de hardware.
*   **Ecossistema Open Source:** Ubuntu, Debian e Fedora.

### 🔹 Aula 02: Operação e Configuração do Windows
*   **Interface Gráfica (GUI):** Ferramentas administrativas, serviços e controle de permissões.

### 🔹 Aula 03: Operação de Sistemas via CLI (Interface de Linha de Comando)
*   **Comandos Essenciais:** Navegação e gerenciamento de arquivos via terminais (CMD e Bash).

### 🔹 Aula 04: Automatização e Introdução a Variáveis
*   **Scripts:** Criação e atribuição de variáveis de ambiente para execução em lote.

### 🔹 Aula 05: Fundamentos de Segurança Cibernética
*   **Proteção Ativa:** Configuração de firewalls, controle de malwares e princípios de privilégio mínimo.
